تطبيق كتاب الهدى

آخر تحديث: ١٦ يونيو ٢٠٢٦

تحترم كتاب الهدى خصوصيتك وتلتزم بحماية أي معلومات قد تشاركها أثناء استخدام التطبيق. توضح هذه السياسة كيفية تعاملنا مع بياناتك.

١. البيانات التي نجمعها

لا نجمع أي بيانات شخصية. لا يتطلب التطبيق إنشاء حساب، ولا يسجل اسمك، بريدك الإلكتروني، رقم هاتفك، موقعك الجغرافي، أو أي معلومات تعريفية.

  • لا تتبع، لا تحليلات، لا إعلانات.
  • لا نستخدم Google Analytics أو Firebase أو أي أدوات تتبع.
  • نستخدم Meta (Facebook) Graph API فقط لتحميل ملفات الصوت عبر رابط مؤقت — لا يتم إرسال أي بيانات عنك إلى Meta.

٢. كيفية عمل البث الصوتي

يعتمد التطبيق على روابط مؤقتة من Meta (Facebook) لبث القرآن الكريم. التطبيق يطلب رابطاً مؤقتاً للملف الصوتي عبر Graph API باستخدام معرف رسالة (message ID) مخزن محلياً — لا يتم مشاركة أي معرّف جهاز أو بيانات شخصية مع Meta.

٣. التخزين المحلي

يحتفظ التطبيق بمعلومات محلية على جهازك فقط:

  • تفضيلاتك (القارئ المفضل، سرعة التشغيل، قوائم التشغيل) — عبر SharedPreferences.
  • سجل الاستماع — عبر قاعدة بيانات Room محلية.
  • ذاكرة مؤقتة للروابط — ٤٥ دقيقة فقط لتحسين الأداء.

كل هذه البيانات تبقى على جهازك ولا تُرسل إلى أي خادم خارجي.

٤. التخزين المؤقت (Cache)

الروابط المؤقتة تُخزَّن محلياً لمدة ٤٥ دقيقة لتجنب الطلبات المتكررة إلى Graph API. بعد انتهاء المدة، تُمسح تلقائياً.

٥. أمان الاتصال

جميع الاتصالات مع Graph API وروابط البث تتم عبر HTTPS. المفاتيح المستخدمة في التشفير موزعة بين الكود المصدري والملفات المحمية.

٦. خدمات الطرف الثالث

التطبيق لا يستخدم أي خدمات طرف ثالث للإعلانات أو التحليلات. الخدمة الوحيدة المستخدمة هي Meta Graph API لغرض تشغيل الصوت فقط.

٧. حذف البيانات

بما أن جميع البيانات محلية، يمكنك حذفها ببساطة عبر:

  • مسح بيانات التطبيق من إعدادات جهازك (الإعدادات ← التطبيقات ← كتاب الهدى ← مسح البيانات).
  • أو إلغاء تثبيت التطبيق بالكامل.

٨. الأطفال

لا يجمع التطبيق بيانات من أي مستخدم بغض النظر عن العمر. يمكن استخدامه بأمان من قبل جميع الفئات العمرية.

٩. التعديلات على سياسة الخصوصية

قد نقوم بتحديث هذه السياسة عند الحاجة. سنشير إلى تاريخ آخر تحديث في أعلى الصفحة. ننصح بمراجعة هذه الصفحة دورياً.
